## How to Use

### Local Development

1. Clone or download this folder
2. Open `index.html` in your web browser
3. The registration page will appear first
4. Enter your name to access the portfolio

### Clear Registration

To reset the registration and see the registration page again:
- Open browser console (F12)
- Type: `localStorage.clear()`
- Refresh the page

## Deployment Options

### Option 1: GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from the `myportfolio` folder
3. Go to Settings â†’ Pages
4. Select "main" branch as source
5. Your site will be live at: `https://yourusername.github.io/repository-name/`

### Option 2: Vercel

1. Install Vercel CLI: `npm install -g vercel`
2. Navigate to the `myportfolio` folder
3. Run: `vercel`
4. Follow the prompts
5. Your site will be deployed instantly

### Option 3: Netlify

1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `myportfolio` folder
3. Your site will be live in seconds

### Option 4: Any Web Hosting

Simply upload all files to your web hosting via FTP:
- Upload to public_html or www folder
- Make sure `index.html` is in the root
- Access via your domain name

## Customization

### Change Colors

Edit the CSS variables in `css/style.css`:

```css
:root {
    --primary-color: #3b82f6;
    --primary-dark: #2563eb;
    --bg-dark: #111827;
    --bg-darker: #0f172a;
    /* ... */
}
```

### Update Images

Replace the Unsplash URLs in `index.html` and `register.html` with your own images.

### Modify Content

- Personal info: Edit text in `index.html`
- Skills & percentages: Update the skills section
- Projects: Add or modify project cards
- Contact info: Update email, location, and links
